# Plan 03-02: Visual Tuning + Final Verification

## Objective

调整 billboard uniform 参数使视觉效果自然，验证最终集成效果。

## Tasks

### 1. 启动开发服务器验证

```bash
cd frontend
npm run dev
```

在浏览器中打开，检查 billboard 是否正确显示 AI 生成图。

### 2. 调参：uBarrelK（弧度）

通过 DebugGUI 或直接修改代码调整 `uBarrelK`：
- 0.1 — 几乎无弧度（接近直线）
- 0.3 — 默认，轻微弧度
- 0.5 — 中等弧度
- 0.8 — 明显弧度

目标：弧形自然，两端向底部消失但不过于夸张。

### 3. 调参：uPlatformHeight（高度）

调整 billboard 占屏幕的比例：
- 0.08 — 底部 8%（很窄的条带）
- 0.12 — 默认，底部 12%
- 0.15 — 底部 15%
- 0.20 — 底部 20%（较宽）

目标：平台高度与树的比例协调，不遮挡树干底部。

### 4. 调参：uPlatformFade（渐隐）

调整底部渐隐宽度：
- 0.01 — 很窄的渐隐（几乎硬切）
- 0.03 — 默认
- 0.05 — 较宽的渐隐

目标：底部边缘自然消失，不突兀。

### 5. 验证视差效果

移动鼠标，确认：
- billboard 保持固定
- 远景 vista 随鼠标轻微浮动
- 两者之间无撕裂或错位

### 6. 验证不同屏幕尺寸

调整浏览器窗口大小，确认：
- 宽屏（16:9）：billboard 居中显示，两侧可能有透明区域
- 窄屏（4:3）：billboard 居中裁剪，显示中间部分
- 极窄（手机竖屏模拟）：只显示纹理中心区域

### 7. 锁定最终参数

将调好的参数值更新到：
- `SdfParamRegistry.ts` 的默认值
- `theme.ts` 的 THEME_DEFAULT

## Verification

手动验证清单：
- [ ] billboard 显示 AI 生成的真实图片
- [ ] 弧形边界自然（中心高、两端向底部消失）
- [ ] 底部渐隐效果平滑
- [ ] 视差分离正确（billboard 固定，远景浮动）
- [ ] 纹理不拉伸（宽高比正确）
- [ ] 与远景色调协调（无明显色差）
- [ ] 不同屏幕尺寸下表现合理

```bash
cd frontend
npx tsc --noEmit
npx vitest run
```
