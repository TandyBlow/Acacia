# Plan 03-01: AI Image Generation + Background Removal

## Objective

使用 gpt-image-1 生成平台 billboard 图片，去背处理后输出为 /public/platform-billboard.png。

## Tasks

### 1. 生成 AI 图片

使用 OpenAI API（gpt-image-1）生成图片：

```bash
# 通过 API 调用生成（具体实现取决于可用工具）
# Prompt 设计要点：
```

**Prompt 核心要素：**
- 视角：wide-angle lens, looking down from a balcony/cliff edge
- 主体：natural stone railing or wooden deck edge with vegetation
- 透视：curved perspective distortion at edges (barrel distortion feel)
- 风格：soft watercolor illustration, warm earth tones, Studio Ghibli-inspired
- 构图：horizontal panoramic, subject fills bottom 60% of frame, sky/background in top 40%
- 尺寸：2048x682 (3:1)

**Prompt 示例：**
```
A wide-angle view looking down from a stone balcony edge into a lush valley below. 
The balcony railing is made of weathered natural stone with moss and small plants growing in cracks. 
Soft watercolor illustration style with warm earth tones. 
The perspective shows gentle barrel distortion - the railing curves slightly at the edges. 
Horizontal panoramic composition, 3:1 aspect ratio. 
Studio Ghibli-inspired art style, soft lighting, peaceful atmosphere.
```

### 2. 去背处理

**方案 A（推荐）：rembg**
```bash
pip install rembg
rembg i input.png output.png
```

**方案 B：background-removal-js**
```bash
npx @nicolo-ribaudo/background-removal-js input.png output.png
```

**验证去背质量：**
- 主体轮廓清晰，无明显毛边
- 天空/背景区域完全透明
- 阳台/栏杆边缘保留完整

### 3. 输出文件

将去背后的 PNG 放置到：
```
frontend/public/platform-billboard.png
```

确认文件：
- 格式：PNG with alpha channel
- 尺寸：2048x682（或接近 3:1）
- 文件大小：< 2MB（合理的 web 资源大小）

## Verification

```bash
# 验证文件存在且有 alpha 通道
file frontend/public/platform-billboard.png
# 验证尺寸
identify frontend/public/platform-billboard.png 2>/dev/null || python3 -c "from PIL import Image; img=Image.open('frontend/public/platform-billboard.png'); print(img.size, img.mode)"
```

- 文件存在
- 有 alpha 通道（RGBA mode）
