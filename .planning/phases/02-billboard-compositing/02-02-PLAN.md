# Plan 02-02: Shader Compositing Logic + Parallax + Tests

## Objective

在 backgroundRaymarch.ts 的 main() 末尾实现 billboard 合成逻辑（桶形畸变 + 纹理采样 + alpha 混合 + 视差分离），并补充单元测试。

## Tasks

### 1. 实现视差分离

**File:** `frontend/src/components/tree/shaders/backgroundRaymarch.ts`

当前 main() 中的视差偏移（line ~147）：
```glsl
uv.x += (uMouseUV.x - 0.5) * 0.06;
```

这个偏移只应用于 vista 的 ray direction。billboard 采样需要用原始 `vScreenUV`（不含偏移）。当前实现已经满足这个要求——`vScreenUV` 是 varying，不受 `uv` 变量修改影响。验证 billboard 合成代码使用 `vScreenUV` 而非 `uv`。

### 2. 在 main() 末尾添加 billboard 合成逻辑

**File:** `frontend/src/components/tree/shaders/backgroundRaymarch.ts`

在 `gl_FragColor = vec4(col, 1.0);` 之前插入：

```glsl
// Billboard compositing (barrel distortion arc)
vec2 c = vScreenUV - 0.5;
float r2 = dot(c, c);
float distort = 1.0 + uBarrelK * r2;
vec2 distortedUV = 0.5 + c * distort;

if (distortedUV.y < uPlatformHeight) {
  float uvX = 0.5 + (vScreenUV.x - 0.5) * uResolution.x / uPlatformTexWidth;
  float uvY = distortedUV.y / uPlatformHeight;
  
  if (uvX >= 0.0 && uvX <= 1.0) {
    vec4 platformSample = texture2D(uPlatformTexture, vec2(uvX, uvY));
    float edgeFade = smoothstep(0.0, uPlatformFade, distortedUV.y);
    float alpha = platformSample.a * edgeFade;
    col = mix(col, platformSample.rgb, alpha);
  }
}
```

注意：将 `bool inBillboard` 和 `float(...)` 乘法改为 if 分支，避免在非 billboard 区域做无用的 texture2D 采样。

### 3. 处理 sky early return

当前 main() 有一个 early return：
```glsl
if (rd.y > 0.25) {
  vec3 col = mix(uSkyBottomColor, uSkyTopColor, vScreenUV.y);
  gl_FragColor = vec4(col, 1.0);
  return;
}
```

这个 early return 在 billboard 合成逻辑之前。如果 billboard 区域的像素恰好 rd.y > 0.25（不太可能，因为 billboard 在底部），需要确保 billboard 仍然被合成。

解决方案：将 early return 改为设置 `col` 并跳过 raymarch，但不 return，让 billboard 合成逻辑仍然执行：

```glsl
bool isSky = rd.y > 0.25;
vec3 col;

if (isSky) {
  col = mix(uSkyBottomColor, uSkyTopColor, vScreenUV.y);
} else {
  // raymarch loop...
  // shading...
}

// Billboard compositing (always runs)
// ...

gl_FragColor = vec4(col, 1.0);
```
