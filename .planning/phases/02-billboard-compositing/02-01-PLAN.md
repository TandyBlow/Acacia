# Plan 02-01: Rename Texture Uniform + Register New Uniforms

## Objective

将 uCliffTexture 重命名为 uPlatformTexture，注册 4 个新 billboard uniform。

## Tasks

### 1. 全局替换 uCliffTexture → uPlatformTexture

搜索范围：
- `frontend/src/components/tree/shaders/backgroundRaymarch.ts` — shader 声明
- `frontend/src/components/tree/scene/BackgroundRenderer.ts` — uniform 对象 + TextureLoader 回调

```bash
grep -rn "uCliffTexture" frontend/src/components/tree/ --include="*.ts"
```

替换所有出现。

### 2. 注册新 uniforms

**File:** `frontend/src/components/tree/scene/SdfParamRegistry.ts`

在 registry 中添加 4 个新条目：

| GLSL name | TS key | Type | Default |
|-----------|--------|------|---------|
| uBarrelK | bgBarrelK | float | 0.3 |
| uPlatformHeight | bgPlatformHeight | float | 0.12 |
| uPlatformFade | bgPlatformFade | float | 0.03 |
| uPlatformTexWidth | bgPlatformTexWidth | float | 2048.0 |

### 3. 在 backgroundRaymarch.ts 的 shader 字符串中声明新 uniforms

由于使用 `generateGlslUniforms()` 自动生成 uniform 声明，注册到 SdfParamRegistry 后应自动出现在 shader 中。验证 `generateGlslUniforms()` 输出包含这 4 个新 uniform。

### 4. 在 TreeStyleParams 类型中添加对应字段

**File:** `frontend/src/constants/theme.ts`

在 TreeStyleParams interface 中添加：
- bgBarrelK: number
- bgPlatformHeight: number
- bgPlatformFade: number
- bgPlatformTexWidth: number

在 THEME_DEFAULT 中添加默认值。

### 5. 更新纹理加载路径

**File:** `frontend/src/components/tree/scene/BackgroundRenderer.ts`

将 `'/generated-cliff-texture.png'` 改为 `'/platform-billboard.png'`。

## Verification

```bash
cd frontend
npx tsc --noEmit 2>&1 | tail -10
npx vitest run --reporter=verbose 2>&1 | tail -20
grep -rn "uCliffTexture" src/ --include="*.ts"  # 应无结果
```
